# Frontend Design Skill (Adapted)

## Overview
This skill focuses on creating "wow" factor UIs. It prioritizes:
- **Visual Excellence**: Curated color palettes, dark modes, and typography.
- **Micro-animations**: Subtle hovers and transitions.
- **Clean Structure**: Semantic HTML and flexible CSS.

## Guidelines
1. **No Defaults**: Avoid standard browser colors. Use HSL or design system tokens.
2. **Typography**: Use Google Fonts or premium font pairings.
3. **Responsiveness**: Design mobile-first.
4. **Wow Elements**: Use gradients, glassmorphism, or modern shadows.

## Patterns
- **Glassmorphism**: `background: rgba(255, 255, 255, 0.1); backdrop-filter: blur(10px);`
- **Modern Gradients**: Soft, multi-hue transitions.
